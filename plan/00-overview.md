## 底部导航栏架构

```
MainShell (Scaffold)
├── IndexedStack（保持Tab状态）
│   ├── [0] HomePage      - 首页（记录、今日概览、周报摘要、FAB记录按钮）
│   ├── [1] StatsPage     - 数据（周报入口、月报入口、年报入口）
│   ├── [2] RankingPage   - 排行（全球/同城/好友 TabBar）
│   └── [3] SettingsPage  - 设置（个人档案、安全、数据管理、关于）
└── BottomNavigationBar
    ├── 🏠 首页  (home_outlined / home)
    ├── 📊 数据  (bar_chart_outlined / bar_chart)
    ├── 🏆 排行  (leaderboard_outlined / leaderboard)
    └── ⚙️ 设置  (settings_outlined / settings)
```

### 子页面导航规则
- 从 Tab 内 Navigator.pushNamed() 打开的子页面（如周报、记录详情、安全页等）会覆盖整个 MainShell（含底部栏隐藏）
- 返回时自动恢复底部栏
- IndexedStack 保持各 Tab 滚动位置与状态

```
状态管理(Riverpod) ─────┬──→ 首页记录模块

数据持久化层 ─────────────┴──→ 后端API集成

                              反作弊系统  成就系统

MainShell (底部导航) ───────→ 整合首页/数据/排行/设置 四大Tab
```
